# rcg
react components generator from template file.

## install

```
npm i rcg -D
```

## usage

```
rcg name templatePath
```


## option

### template

template path


### props (string)

react path

### prefix (string)

prefix

#### dest (string)

output directory path

## package.json

#### root template path

```json
  "rcg": {
    "templates": "./templates/"
  },
```
